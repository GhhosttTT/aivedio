# Git 忽略规则说明

## 已忽略的文件和目录

### 1. AI 模型文件（不上传）
- `models/` - 所有模型文件
- `*.gguf` - GGUF格式模型
- `*.safetensors` - SafeTensors格式模型
- `*.bin` - 二进制模型文件
- `*.pth` / `*.pt` - PyTorch模型文件
- `*.ckpt` - Checkpoint文件
- `*.onnx` - ONNX格式模型

### 2. 生成的媒体文件（不上传）
- `storage/` - 所有生成的图片、视频、音频
- `output/` - 输出目录
- `results/` - 结果目录
- `*.mp4`, `*.avi`, `*.mov` - 视频文件
- `*.mp3`, `*.wav`, `*.ogg` - 音频文件
- `*.png`, `*.jpg`, `*.jpeg`, `*.gif`, `*.webp` - 图片文件

### 3. ComfyUI 目录（不上传）
- `ComfyUI/` - 整个ComfyUI目录（包含模型和大文件）

### 4. 前端依赖（不上传）
- `frontend/node_modules/` - Node.js依赖包
- `frontend/dist/` - 构建输出
- `frontend/build/` - 构建输出

### 5. 日志和数据库（不上传）
- `logs/` - 日志文件
- `*.log` - 所有日志文件
- `*.db`, `*.sqlite` - 数据库文件

### 6. Python缓存和临时文件（不上传）
- `__pycache__/` - Python字节码缓存
- `*.pyc`, `*.pyo` - Python编译文件
- `.pytest_cache/` - Pytest缓存
- `.venv/`, `venv/` - 虚拟环境

### 7. 环境变量（不上传）
- `.env` - 环境变量配置文件
- `.env.local` - 本地环境变量
- `.env.production` - 生产环境变量

## 如何下载模型

由于模型文件没有上传到Git，你需要手动下载：

```bash
# 下载所有需要的模型
bash scripts/download_models.sh
```

或者在Windows上使用PowerShell：

```powershell
# 下载所有需要的模型
.\scripts\download_models.ps1
```

## 如何安装ComfyUI

ComfyUI目录也没有上传，你需要单独克隆或下载：

```bash
# 方法1：克隆ComfyUI官方仓库
cd aivedio
git clone https://github.com/comfyanonymous/ComfyUI.git

# 方法2：使用提供的安装脚本
bash scripts/setup_comfyui.sh
```

## 上传说明

本项目只包含源代码和配置文件，不包含：
- AI模型文件（通常很大，几十GB）
- 生成的图片、视频、音频文件
- ComfyUI完整安装（需要单独配置）
- 数据库文件
- 日志文件

要使用本项目，请按照 `README.md` 中的安装指南进行操作。
