# RTX 3060 性能优化指南

## 💻 硬件配置分析

### RTX 3060 规格
- **显存**: 12GB GDDR6
- **CUDA 核心**: 3584
- **性能定位**: 中端显卡

### 能跑 SDXL 吗?

**✅ 可以跑!** 但需要优化配置

## 📊 性能评估

### SDXL 模型内存需求

| 配置 | 显存占用 | RTX 3060 能跑? |
|------|----------|----------------|
| **基础 SDXL** | 6-8GB | ✅ 可以 |
| **SDXL + LoRA** | 7-9GB | ✅ 可以 |
| **SDXL + IP-Adapter** | 8-10GB | ✅ 勉强可以 |
| **SDXL + Refiner** | 10-12GB | ⚠️ 接近极限 |
| **多个模型同时加载** | 12GB+ | ❌ 不行 |

### 生成速度预估

| 分辨率 | 步数 | 预计时间 |
|--------|------|----------|
| 1024x576 | 20 | ~15-20秒 |
| 1024x576 | 30 | ~20-30秒 |
| 1280x720 | 30 | ~30-40秒 |
| 1280x720 | 40 | ~40-60秒 |

## ⚠️ 可能遇到的问题

### 1. 显存不足 (OOM)

**症状**:
```
RuntimeError: CUDA out of memory
```

**原因**:
- 分辨率太高
- 步数太多
- 同时加载多个模型
- IP-Adapter + Refiner 同时使用

### 2. 生成速度慢

**症状**:
- 每张图需要 1-2 分钟
- 系统卡顿

**原因**:
- 参数设置过高
- 后台程序占用显存
- 没有优化配置

## 🎯 优化方案

### 方案 1: 基础优化 (必做!)

#### 1.1 降低分辨率

```python
# src/tasks/image_tasks.py

# ❌ 过高 (可能 OOM)
width = 1280
height = 720

# ✅ 推荐 (RTX 3060)
width = 1024
height = 576
```

#### 1.2 适度降低步数

```python
# ❌ 过高 (太慢)
steps = 40

# ✅ 推荐 (RTX 3060)
steps = 25-28
```

#### 1.3 优化 CFG

```python
# ✅ 推荐
cfg_scale = 6.0  # 既快又自然
```

### 方案 2: ComfyUI 优化

#### 2.1 启用低显存模式

编辑 ComfyUI 启动脚本:

```bash
# Windows
python main.py --lowvram

# 或者更激进的优化
python main.py --medvram --opt-split-attention
```

**参数说明**:
- `--lowvram`: 低显存模式 (8-10GB 显存)
- `--medvram`: 中等显存模式 (6-8GB 显存)
- `--opt-split-attention`: 优化注意力机制,降低显存

#### 2.2 禁用不必要的功能

编辑 `configs/comfyui_workflow_sdxl.json`:

```json
{
  "optimization": {
    "enable_refiner": false,  // 禁用 Refiner (节省 2-3GB)
    "enable_upscale": false,  // 禁用高清放大 (节省 1-2GB)
    "vae_tiling": true        // 启用 VAE 分块 (降低显存)
  }
}
```

## 🔍 故障排查

### 问题 1: CUDA Out of Memory

**解决方案**:
1. 降低分辨率 (1024x576)
2. 降低步数 (20-25)
3. 禁用 Refiner
4. 使用 `--lowvram` 启动
5. 关闭其他程序

### 问题 2: 生成速度太慢

**解决方案**:
1. 降低步数 (20-25)
2. 使用更快的采样器 (DPM++ 2M)
3. 降低分辨率
4. 更新 NVIDIA 驱动
5. 检查 GPU 温度 (可能降频)

### 问题 3: 质量不够好

**解决方案**:
1. 使用 RealVisXL V4.0 模型
2. 适当提高步数 (28-30)
3. 优化提示词
4. 使用 Detail Tweaker LoRA
5. 调整 CFG (6.0-6.5)
