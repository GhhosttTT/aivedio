# 角色外貌特征设置指南

## 快速开始

### 新项目（推荐）

创建新项目时，系统会**自动**为每个角色生成独特的外貌特征，无需手动设置！

```bash
# 1. 创建项目
# 2. 生成剧本
# 3. 系统自动为角色生成外貌特征
# 4. 开始生产，角色将自动区分
```

### 现有项目

如果你的项目已经创建，但角色还没有外貌特征，可以使用工具手动设置：

```bash
python scripts/setup_character_appearances.py
```

## 为什么需要角色外貌特征？

在短剧制作中，如果不设置角色外貌特征，会出现以下问题：

❌ **问题1：所有男性角色长一个样子**
- 主角、反派、配角的图像都很相似
- 观众无法区分不同角色

❌ **问题2：角色在不同分镜中不一致**
- 同一角色在不同场景中外貌变化很大
- 影响观看体验

✅ **设置外貌特征后**：
- 每个角色有独特的视觉特征
- 主角和反派明显区分
- 角色在不同分镜中保持一致

## 外貌特征包含什么？

一个完整的角色外貌特征应该包含：

### 1. 基本信息
- **年龄段**：25岁、30岁、45岁等
- **性别**：男性、女性

### 2. 面部特征
- **发型**：短发、长发披肩、中分发型、微卷短发
- **发色**：黑发、棕发、花白短发
- **脸型**：瓜子脸、方脸、圆脸
- **独特标识**：戴金丝眼镜、略带胡茬、精致妆容

### 3. 身材特征
- 身材苗条、身材挺拔、保养得当

### 4. 服装风格
- 商务西装、优雅连衣裙、休闲装、旗袍、中式唐装

### 5. 气质特征
- 温柔气质、霸道总裁气场、成熟稳重、不羁气质、邻家女孩感

## 常见问题

### Q1: 我的项目已经生成了图像，现在设置外貌特征还有用吗？

**A**: 有用！设置外貌特征后，重新生成图像时会使用新的外貌特征。建议：
1. 设置好所有角色的外貌特征
2. 删除已生成的图像
3. 重新开始生产任务

### Q2: 自动生成的外貌特征不满意怎么办？

**A**: 使用工具手动调整：
```bash
python scripts/setup_character_appearances.py
```
选择"交互式设置"，然后为角色重新设置外貌特征。

### Q3: 如何确保主角和反派有明显区分？

**A**: 在设置外貌特征时，注意以下几点：
- **年龄差异**：主角30岁，反派35岁
- **发型差异**：主角短发，反派中分发型
- **气质差异**：主角成熟稳重，反派不羁冷峻
- **服装差异**：主角商务西装，反派休闲西装
- **独特标识**：主角戴眼镜，反派有胡茬

### Q4: 角色外貌特征会影响 IP-Adapter 吗？

**A**: 不会冲突，反而会增强效果：
- **外貌特征**：确保不同角色有区分
- **IP-Adapter**：确保同一角色在不同分镜中一致
- **两者结合**：既有区分度，又有一致性

### Q5: LLM 服务不可用时怎么办？

**A**: 系统会自动降级：
- 新项目：使用角色描述作为外貌特征
- 现有项目：使用工具手动设置外貌特征

## 最佳实践

### 1. 提前规划角色外貌

在创建项目前，先规划好主要角色的外貌特征：

| 角色 | 年龄 | 发型 | 服装 | 气质 | 独特标识 |
|------|------|------|------|------|----------|
| 女主 | 25岁 | 长发披肩 | 优雅连衣裙 | 温柔 | 清秀面容 |
| 男主 | 30岁 | 短发 | 商务西装 | 成熟稳重 | 戴金丝眼镜 |
| 反派 | 35岁 | 中分发型 | 休闲西装 | 不羁 | 略带胡茬 |

### 2. 确保主要角色差异明显

- 主角和反派的外貌特征差异要大
- 至少在3个维度上有明显区别（年龄、发型、气质）

### 3. 配角可以简化

- 配角的外貌特征可以相对简单
- 但仍需要有基本的区分度

### 4. 定期检查

- 在生成图像后，检查角色是否有明显区分
- 如果不满意，调整外貌特征后重新生成
