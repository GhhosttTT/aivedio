# 增强版剧本生成方案

## 问题分析

当前系统生成的剧本存在以下问题：

1. **剧本不够详尽**：LLM 生成的剧本只包含基本的场景描述，缺乏完整的故事结构
2. **缺少章节概念**：没有像书籍一样的章节划分，导致剧情连贯性不足
3. **分镜描述过于简单**：每个分镜的描述字数不足，导致图像生成质量不稳定
4. **缺少故事大纲**：没有先生成完整的故事大纲再细化到分镜

## 改进方案

### 1. 分层生成策略

采用类似书籍的结构，分层组织剧本：

```
完整剧本
├── 故事大纲（200-300字）
│   ├── 背景设定
│   ├── 主要冲突
│   ├── 情节发展
│   └── 结局走向
│
├── 角色设定（每个角色100+字）
│   ├── 年龄性别
│   ├── 外貌特征（详细）
│   ├── 服装风格（详细）
│   ├── 性格特点
│   └── 角色定位
│
├── 第一章：[章节标题]
│   ├── 章节概述（100-150字）
│   ├── 分镜 1（详细描述，200+字）
│   │   ├── 章节位置
│   │   ├── 故事节点
│   │   ├── 环境描述（50+字）
│   │   ├── 人物描述（40+字）
│   │   ├── 镜头描述（20+字）
│   │   ├── 光线描述（20+字）
│   │   ├── 氛围描述（20+字）
│   │   ├── 对话
│   │   └── 情感
│   ├── 分镜 2（详细描述）
│   └── ...
│
├── 第二章：[章节标题]
│   ├── 章节概述
│   ├── 分镜 X
│   └── ...
│
└── 第三章：[章节标题]
    ├── 章节概述
    ├── 分镜 Y
    └── ...
```

### 2. 详细描述要求

#### 2.1 故事大纲（200-300字）
- 背景设定：时代、地点、社会环境
- 主要冲突：核心矛盾是什么
- 情节发展：故事如何推进
- 结局走向：最终如何解决

#### 2.2 角色设定（每个角色100+字）
- **年龄性别**：例如"25岁，男性"
- **外貌特征**：详细描述五官、发型、身材、肤色等
  - 示例："浓眉大眼，黑色短发，身材修长，皮肤白皙，眼神深邃"
- **服装风格**：详细描述服装款式、颜色、材质、配饰
  - 示例："黑色修身西装，白色衬衫，银色领带夹，黑色皮鞋，手腕佩戴银色手表"
- **性格特点**：例如"冷静理性，不善言辞，内心温柔"
- **角色定位**：例如"男主角，公司高管"

#### 2.3 章节概述（100-150字）
- 本章的主要情节
- 情感基调
- 关键转折点
- 与整体故事的关系

#### 2.4 分镜详细描述（每个分镜200+字）

**章节位置**：例如"第一章 开场"

**故事节点**：例如"男主角独自在办公室加班，展现他的工作状态和孤独感"

**环境描述（至少50字）**：
- 时间：具体时间点
- 地点：详细的空间信息
- 天气：天气状况
- 光线：自然光或人造光
- 具体物品：至少3个可见物品
- 背景元素：远景、中景、近景的元素
- 空间布局：物体的相对位置

示例：
> "深夜11点的现代化办公室，落地窗外是璀璨的城市夜景，室内只有一盏台灯亮着，桌上堆满文件和咖啡杯，空调发出低沉的嗡嗡声，墙上的时钟指针缓慢移动，书架上整齐摆放着各类书籍，地面是深色木地板，反射着微弱的灯光"

**人物描述（至少40字）**：
- 位置：角色在场景中的位置
- 姿势：身体姿态
- 表情：面部表情细节
- 动作：正在做什么
- 服装细节：当前穿着
- 肢体语言：传达的情绪
- 情绪状态：内心状态

示例：
> "男主角坐在黑色真皮办公椅上，身体微微前倾，左手撑着额头，右手握着钢笔在文件上快速书写，眉头紧锁，眼神专注而疲惫，白色衬衫的袖子卷到手肘，领带松开挂在脖子上，整个人散发出疲惫但专注的气质"

**镜头描述（至少20字）**：
- 景别：特写/中景/全景
- 拍摄角度：正面/侧面/背面，俯视/平视/仰视
- 构图：三分法、对称、黄金分割等
- 焦点：焦点在哪里
- 景深：浅景深/深景深

示例：
> "中景镜头，从侧前方30度角拍摄，人物位于画面右侧三分之一处，左侧是虚化的办公室背景，焦点在人物脸部，景深较浅，营造电影感"

**光线描述（至少20字）**：
- 光源类型：自然光/人造光
- 光源位置：从哪个方向照射
- 色温：暖色/冷色
- 明暗对比：高对比/低对比
- 阴影效果：硬阴影/软阴影

示例：
> "主光源是台灯的暖黄色光线从左侧照射，在脸部形成柔和的明暗对比，背景的城市灯光透过落地窗形成冷色调的轮廓光，整体呈现暖冷对比的视觉效果，高对比度"

**氛围描述（至少20字）**：
- 情绪基调：孤独/温暖/紧张/浪漫等
- 视觉风格：电影级/纪实/梦幻等
- 色彩倾向：暖色调/冷色调/高饱和/低饱和
- 质感：细腻/粗糙/柔和/锐利
- 电影感：是否有电影质感

示例：
> "孤独、专注、疲惫的氛围，电影级画质，暖黄与冷蓝的色彩对比，高对比度，细腻的光影层次，真实感强，类似《社交网络》的视觉风格"

### 3. 实施步骤

#### 步骤 1：修改 LLM Prompt 模板

在 `src/services/llm_service.py` 的 `generate_script_prompt` 方法中：

1. 增加 `num_chapters` 参数（默认 3 章）
2. 计算每章分镜数量：`scenes_per_chapter = num_scenes // num_chapters`
3. 使用增强版 Prompt 模板（见下方）

#### 步骤 2：增强剧本解析逻辑

在 `src/services/script_generator.py` 的 `parse_script` 方法中：

1. 解析【故事大纲】部分
2. 解析【角色】部分（增强版，包含更多细节）
3. 解析【章节】结构
4. 解析每个分镜的详细描述（包含新增的字段）

#### 步骤 3：数据库模型扩展

在 `src/database/models.py` 中：

1. 增加 `Chapter` 模型（章节表）
2. 扩展 `Scene` 模型，增加字段：
   - `chapter_id`：所属章节
   - `chapter_position`：在章节中的位置
   - `story_node`：故事节点描述
   - `environment_detail`：详细环境描述
   - `character_detail`：详细人物描述
   - `camera_detail`：详细镜头描述
   - `lighting_detail`：详细光线描述
   - `atmosphere_detail`：详细氛围描述

#### 步骤 4：前端展示优化

在前端界面中：

1. 增加章节导航（类似书籍目录）
2. 显示故事大纲
3. 显示章节概述
4. 分镜卡片显示更多细节
5. 支持按章节筛选和查看

### 4. 增强版 Prompt 模板

```python
def generate_script_prompt(
    self,
    theme: Optional[str] = None,
    outline: Optional[str] = None,
    num_scenes: int = 10,
    num_characters: int = 2,
    style: str = "现代都市",
    num_chapters: int = 3
) -> str:
    """
    构建剧本生成的 Prompt（支持章节结构）
    
    Args:
        theme: 主题关键词（可选）
        outline: 故事大纲（可选）
        num_scenes: 分镜数量（默认 10）
        num_characters: 角色数量（默认 2）
        style: 风格偏好（默认"现代都市"）
        num_chapters: 章节数量（默认 3）
        
    Returns:
        完整的 Prompt 文本
    """
    if not theme and not outline:
        raise ValueError("主题和大纲至少需要提供一个")
    
    # 计算每章分镜数量
    scenes_per_chapter = num_scenes // num_chapters
    
    # 构建增强版 Prompt 模板
    prompt_template = """你是一位专业的短剧编剧和视觉导演。请根据以下要求创作一个完整的短剧剧本，像写书一样分章节组织：

主题：{theme}
故事大纲：{outline}
总分镜数量：{num_scenes}
章节数量：{num_chapters}（每章约 {scenes_per_chapter} 个分镜）
角色数量：{num_characters}
风格：{style}

请按照以下格式输出：

【故事大纲】
（200-300字的完整故事概述，包含：背景设定、主要冲突、情节发展、结局走向）

【角色】
- 角色1：[姓名]
  * 年龄性别：[例如：25岁，男性]
  * 外貌特征：[详细描述五官、发型、身材、肤色等，例如：浓眉大眼，黑色短发，身材修长，皮肤白皙]
  * 服装风格：[详细描述服装款式、颜色、材质、配饰，例如：黑色修身西装，白色衬衫，银色领带夹，黑色皮鞋]
  * 性格特点：[例如：冷静理性，不善言辞，内心温柔]
  * 角色定位：[例如：男主角，公司高管]

【第一章：[章节标题]】
章节概述：[100-150字，描述本章的主要情节、情感基调、关键转折点]

分镜1：
- 章节位置：第一章 开场
- 故事节点：[例如：男主角独自在办公室加班，展现他的工作状态和孤独感]
- 环境描述：[至少50字，详细描述时间、地点、天气、光线、具体物品、背景元素、空间布局]
- 人物描述：[至少40字，详细描述角色的位置、姿势、表情、动作、服装细节、肢体语言]
- 镜头描述：[详细描述拍摄角度、景别、构图、焦点]
- 光线描述：[详细描述光源类型、位置、色温、明暗对比、阴影效果]
- 氛围描述：[详细描述情绪基调、视觉风格、色彩倾向、质感]
- 出现角色：[角色1]
- 对话：（如果有对话，写在这里；如果没有对话，写"无"）
- 说话人：（如果有对话，标注说话人；如果没有对话，写"无"）
- 情感：[例如：疲惫、专注、孤独]

重要要求：
1. 【故事大纲】必须完整，200-300字，涵盖整个故事的起承转合
2. 【角色】描述必须极其详细，包含所有外貌、服装、性格细节，确保角色一致性
3. 【章节】结构清晰，每章有独立的情节推进和情感基调
4. 【分镜】描述必须达到以下字数要求：
   - 环境描述：至少50字
   - 人物描述：至少40字
   - 其他描述：至少20字
5. 每个分镜必须包含"故事节点"，说明它在整体故事中的作用
6. 所有描述都要详细具体，达到"可视化小说"的水平
7. 分镜之间要有连贯性，避免场景跳跃过大
8. 每章结尾要有情节转折或情感高潮
"""
    
    # 填充模板
    prompt = prompt_template.format(
        theme=theme or "未指定",
        outline=outline or "未指定",
        num_scenes=num_scenes,
        num_chapters=num_chapters,
        scenes_per_chapter=scenes_per_chapter,
        num_characters=num_characters,
        style=style
    )
    
    return prompt
```

### 5. 预期效果

实施此方案后，预期达到以下效果：

1. **剧本更完整**：有完整的故事大纲和章节结构
2. **描述更详细**：每个分镜的描述字数从 50-100 字增加到 200+ 字
3. **图像质量更稳定**：详细的描述让 AI 绘画能更准确地理解场景
4. **剧情更连贯**：章节结构确保故事有清晰的起承转合
5. **角色更一致**：详细的角色设定确保同一角色在不同分镜中保持一致

**高优先级（立即实施）**：
1. 修改 LLM Prompt 模板（增加章节结构和详细描述要求）

**中优先级（后续实施）**：

### 7. 测试验证

实施后，通过以下方式验证效果：

1. **生成测试**：使用相同主题生成剧本，对比新旧版本的详细程度
2. **图像质量测试**：对比使用新旧剧本生成的图像质量
3. **用户反馈**：收集创作者对剧本完整性和图像质量的反馈
4. **字数统计**：统计每个分镜的描述字数，确保达到要求

### 8. 注意事项

1. **LLM 性能**：更详细的 Prompt 会增加 LLM 的生成时间和 token 消耗
2. **解析复杂度**：更复杂的格式需要更健壮的解析逻辑
3. **向后兼容**：保留对旧格式剧本的支持
4. **用户体验**：确保生成过程有清晰的进度提示

## 总结

通过采用分层生成策略和详细描述要求，可以显著提升剧本的完整性和图像生成质量。这个方案的核心思想是：**把剧本当作一本书来写，而不是简单的分镜列表**。

关键改进点：
- ✅ 增加故事大纲（200-300字）
- ✅ 增加章节结构（3-5章）
- ✅ 增加章节概述（100-150字/章）
- ✅ 增加分镜详细描述（200+字/分镜）
- ✅ 增加故事节点说明
- ✅ 增加角色详细设定（100+字/角色）

这样生成的剧本将更像一部"可视化小说"，为后续的图像和视频生成提供充足的细节支持。
