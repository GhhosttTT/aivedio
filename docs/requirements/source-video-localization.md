# 源片出海 AI 短剧译制功能

## 问题陈述

用户已有中文短剧源片，希望在本地显卡环境下批量生成多语种海外发行版本。核心问题不是重新生成短剧，而是把已有中文源片转换为可发布的多语种字幕成片。

## V1 范围

V1 只做硬字幕版本，不做画面 OCR，不做 AI 配音。

流程：

```text
中文源片
 -> 去字幕/去水印
 -> ASR 识别中文对白
 -> 翻译 9 个语种
 -> 字幕压制
 -> 智能审核
 -> 9 语种成片
```

默认 9 个语种：

- 英语 `en`
- 西班牙语 `es`
- 葡萄牙语 `pt`
- 阿拉伯语 `ar`
- 印尼语 `id`
- 泰语 `th`
- 越南语 `vi`
- 日语 `ja`
- 韩语 `ko`

## 验收标准

- 用户可以上传一个中文源片到项目下。
- 用户可以创建一个出海译制任务并选择目标语种。
- 系统记录任务状态、当前阶段、进度、错误信息。
- 任务阶段固定为：上传、去字幕、ASR、翻译、压制、审核、完成。
- 模型后端未接入时必须明确失败，不允许假装完成。
- 后续接入模型后，每个语种输出一个烧录字幕的 MP4。

## 本地模型建议

| 阶段 | 首选方案 | 备注 |
|---|---|---|
| 去字幕/去水印 | Video Subtitle Remover / ProPainter / LaMa | 这是最大质量风险点 |
| ASR | faster-whisper large-v3 | 适合本地 GPU 批处理 |
| 翻译 | Qwen2.5/3 14B+ 或 API 混合 | 小模型对泰语、阿语、日韩质量不稳 |
| 字幕压制 | FFmpeg + ASS | 需要按语言控制换行、字体、RTL |
| 审核 | 规则词库 + 抽帧多模态 | V1 先文本审核，视频审核后置 |

## 落地顺序

1. 跑通 API 和任务骨架。
2. 接入源片上传与 FFprobe 元数据。
3. 接入 faster-whisper，生成中文字幕 SRT。
4. 接入本地翻译模型，先做英语和西语。
5. 增加 ASS 字幕模板并压制成片。
6. 再接去字幕/去水印模型。
7. 扩展到 9 语种和批量队列。
8. 增加审核报告和人工质检页面。

## 关键风险

- 去字幕/去水印效果不稳定，尤其字幕遮挡人脸或复杂背景时。
- 本地 LLM 翻译质量不一定能稳定达到发行级，需要保留人工校对。
- 阿语 RTL、泰语断词、日韩字体渲染需要单独处理。
- 智能审核不能替代人工终审。
