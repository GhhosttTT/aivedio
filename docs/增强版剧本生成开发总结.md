## 核心改进

### 1. 剧本结构升级

**之前（v2.0）**：
```
【剧本】
【角色】
【分镜】
```

**现在（v3.0）**：
```
【故事大纲】（200-300字）
【角色】（每个角色100+字）
【第一章：章节标题】
  章节概述（100-150字）
  分镜1（200+字）
  分镜2（200+字）
【第二章：章节标题】
  ...
```

### 2. 描述详细度提升

| 描述类型 | v2.0 | v3.0 | 提升幅度 |
|---------|------|------|---------|
| 分镜总字数 | 50-100字 | 200+字 | 2-4倍 |
| 环境描述 | 简单描述 | ≥50字 | 详细化 |
| 人物描述 | 简单描述 | ≥40字 | 详细化 |
| 角色设定 | 简单描述 | 100+字 | 详细化 |

### 3. 新增字段

- **故事大纲**：200-300字的完整故事概述
- **章节结构**：3-5章，每章有独立的情节推进
- **章节概述**：100-150字/章
- **故事节点**：30-50字，说明分镜在故事中的作用
- **章节位置**：例如"第一章 开场"

### 生成时间

## 使用示例

### API 调用示例

```python
# 使用增强版剧本生成
POST /api/projects/{project_id}/generate-script
{
    "theme": "都市爱情",
    "outline": "一个工作狂高管遇到了一个温暖的设计师",
    "num_scenes": 12,
    "num_characters": 2,
    "style": "现代都市",
    "num_chapters": 3,
    "temperature": 0.7,
    "max_tokens": 8192
}
```

### Python 代码示例

```python
from src.services.enhanced_script_prompt import generate_enhanced_script_prompt

# 生成增强版 Prompt
prompt = generate_enhanced_script_prompt(
    theme="都市爱情",
    outline="一个工作狂高管遇到了一个温暖的设计师",
    num_scenes=12,
    num_characters=2,
    style="现代都市",
    num_chapters=3
)

# 使用 LLM 生成剧本
from src.services.llm_service import get_llm_service

llm_service = get_llm_service()
script_text = llm_service.generate(
    prompt=prompt,
    max_tokens=8192,
    temperature=0.7
)
```
