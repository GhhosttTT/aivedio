# 🎨 图像质量优化 - 快速指南

## 问题

生成的图片质量不如 Seeddance 等专业工具

## 解决方案 (3步搞定!)

### 步骤 1: 运行优化脚本 ⚡

```bash
python scripts/optimize_image_quality.py
```

这个脚本会自动:
- ✅ 提高图像分辨率 (1024x576 → 1280x720)
- ✅ 增加采样步数 (20 → 35)
- ✅ 优化 CFG 参数 (7.0 → 8.0)
- ✅ 增强提示词质量 (添加质量标签和细节描述)
- ✅ 优化负面提示词

### 步骤 2: 下载高质量模型 🎯

**推荐模型: Juggernaut XL v9** (最重要!)

1. **下载地址**: https://civitai.com/models/133005/juggernaut-xl
2. **选择版本**: Juggernaut XL v9
3. **下载文件**: juggernaut_xl_v9.safetensors (~6.5GB)
4. **放置位置**: `ComfyUI/models/checkpoints/`

### 步骤 3: 修改配置并重启 🔄

1. **修改 ComfyUI 工作流配置**:
   
   编辑 `configs/comfyui_workflow_sdxl.json`:
   ```json
   {
     "nodes": {
       "CheckpointLoaderSimple": {
         "inputs": {
           "ckpt_name": "juggernaut_xl_v9.safetensors"
         }
       }
     }
   }
   ```

2. **重启服务**:
   ```bash
   # 重启 ComfyUI
   pkill -f comfyui
   cd ComfyUI && python main.py
   
   # 重启 Celery Worker
   pkill -f celery
   celery -A src.tasks.celery_app worker --loglevel=info --pool=solo
   ```

3. **测试**:
   - 创建新项目
   - 生成剧本
   - 开始制作
   - 查看图像质量

## 常见问题 ❓

### Q: 模型文件太大怎么办?
**A**: 使用下载工具 (IDM, aria2) 或分段下载

### Q: 生成速度变慢了?
**A**: 正常现象,高质量需要更多计算。可以降低步数到 30 来加速

### Q: 还是不够好?
**A**: 
1. 确认模型正确加载
2. 增加步数到 40
3. 调整 CFG 到 9.0
4. 查看详细文档: `docs/图像质量优化方案.md`
